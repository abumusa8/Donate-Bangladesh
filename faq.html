What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents web documents as a tree of objects. It allows developers to access and manipulate the structure, content, and style of HTML or XML documents dynamically, enabling interactive web pages.


How do you select an element forom the DOM?

By ID: You can select an element using its unique identifier. By Class Name: You can select elements that share the same class name. By Tag Name: You can select all elements of a specific type, such as all paragraphs or all divs. Using CSS Selectors: You can select elements using more complex queries, like selecting the first element that matches a certain class or selecting all elements that meet specific criteria. These methods allow you to effectively access and manipulate elements on a web page.


What is event delegation in the context of the DOM. and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element rather than to multiple child elements. When an event occurs on a child, it bubbles up to the parent, allowing the parent to handle the event. This approach is useful because it improves performance by reducing the number of event listeners, simplifies code maintenance by centralizing event handling, and easily accommodates dynamic content changes without needing to reattach listeners.


How do you manipulation and element's attributes and styles using the DOM

To manipulate an element's attributes using the DOM, you can use methods like `getAttribute`, `setAttribute`, and `removeAttribute`. These allow you to read, modify, or remove attributes like `id`, `class`, or custom data attributes. For styling, you can directly adjust an element's CSS using the `style` property to change properties like `color`, `background`, and `font-size`. Alternatively, you can manage styles through CSS classes with methods like `classList.add`, `classList.remove`, and `classList.toggle`, which enable you to add, remove, or toggle classes for more organized styling.
